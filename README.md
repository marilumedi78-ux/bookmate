# 📚 BookMate - Tu Compañero de Libros

BookMate es una app de lectura con audio inteligente (TTS), sonidos ambientales y seguimiento de progreso.

## 🚀 Despliegue en Vercel

1. Sube este repositorio a GitHub
2. Ve a [vercel.com](https://vercel.com) y conéctalo
3. Agrega la variable de entorno: `DATABASE_URL=file:./db/custom.db`
4. ¡Despliega!

## 🛠 Tech Stack

- Next.js 16 + TypeScript
- Tailwind CSS 4 + shadcn/ui
- Prisma ORM (SQLite)
- Zustand (state management)
- Framer Motion (animations)
- z-ai-web-dev-sdk (AI features)
